# ToDoアプリ

シンプルなToDo管理アプリです。  
タスクの追加・削除ができます。

## 🔗 URL

https://todo-app-xq8d.onrender.com/

## 🛠 使用技術

- Node.js
- Express
- JavaScript
- HTML / CSS

## ✨ 機能

- タスク追加
- タスク一覧表示
- タスク削除

## 💡 工夫した点

- fetchを使ってフロントとサーバーを連携
- API（GET / POST / DELETE）を実装
- DOM操作で画面を動的に更新

## ⚠️ 注意

※無料プランのため初回アクセス時に時間がかかる場合があります
